# 📚 Anotações - Semana 1 (QA)

## 🧠 Conceitos básicos

### 🔴 Erro (Error)
É um erro humano durante o desenvolvimento do sistema.
Exemplo: desenvolvedor implementa regra incorreta.

---

### 🟠 Falha (Failure)
Ocorre quando o sistema se comporta de forma inesperada devido a um erro.
Exemplo: sistema aceita senha inválida.

---

### 🟢 Bug (Defeito)
É a falha identificada e registrada por um QA.
Exemplo: sistema permite login com senha menor que o esperado.

---

## 🔁 Fluxo
Erro → Falha → Bug

---

## 🛠️ Primeiros testes realizados

- Teste de login com dados válidos
- Teste de login com dados inválidos
- Teste com campos vazios

---

## 📌 Aprendizados

- Importância de documentar testes
- Diferença entre erro, falha e bug
- Pensamento analítico no teste de sistemas
